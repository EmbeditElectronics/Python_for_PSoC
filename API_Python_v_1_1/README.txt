API folder README (8/6/2014)

Copyright 2014, Embedit Electronics

Contains all source code and documentation for the RPiSoC to Python API as of Version 1.0.0

This project was tested using Windows 7 x64 SP1 for programming the RPiSoC with PSoC Creator version 3.0 component pack 7. On the Raspberry Pi, the Raspbian operating system version 2014-01-07 was used. Programs were written with Python 2.7.

Sub-directories:

PSoC Creator:

Contains the PSoC Creator files which should be programmed to the RPiSoC for use of the API. 

Python:

Contains the python modules which comprise the API, as well as a series of example projects which use it.
