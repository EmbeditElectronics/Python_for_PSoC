Detailed Description

This file contains the source code for access or manipulation of all components included in the build. It will do so according to sets of instructions provided by a Master Device.

Version 1.0

Author
Brian Bradley
Bug:
No known bugs, but the VDAC's and IDAC's are currently untested.

Copyright Embedit Electronics
